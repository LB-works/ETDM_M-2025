Testing React Dashboard

If you see this message, the React app is not loading properly.

Common issues:

  • Node.js/npm not installed
  • Dependencies not installed (run npm install)
  • Port already in use
  • Firewall blocking the connection

To run the dashboard:

  1. Open Command Prompt
  2. Navigate to the react-dashboard directory:
    cd "c:\Users\Ledge\Documents\PlatformIO\Projects\Engr Simon project - Client copy\react-dashboard"
  3. Install dependencies (if not already done):
    npm install
  4. Start the development server:
    npm run dev
  5. Open your browser to http://localhost:5173
